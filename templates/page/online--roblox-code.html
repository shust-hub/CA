  • Roblox Code

    Ages: 9-11

    Students will design and code their own creations using Roblox Studio’s immersive educational tools.

    What’s neat about Roblox is that anyone can build a game on it, the platform offers such a vast toolbox which is readily available and welcoming to all skills levels. The programming language used on Roblox is called “Roblox Lua,” a dialect of Lua – which is currently the leading scripting language used in games.

    Level 1: Learn about the difference between Roblox Player and Roblox Studio. Introduction to “Roblox Lua,” and game design.

    Level 2: Why Lua? Students explore the benefits of Lua in game design.

    Level 3: More design and Lua.

    Examples of student projects:

    Creating parkour maps, 2-player games, obstacle course fun
